##   QUIZ-GAME  🎮 ✨

Welcome to QUIZ TIME! 
A simple and fun browser-based quiz game where you can test your knowledge and track your score.

----------

##  Features

- Start screen -welcome message and start button.
- Quiz screen  - shows questions, answers, score, and progress bar.
- Result screen -display  your final score and friendly message.
- Progress bar that fills as you move through the quiz.
- Restart option to play the game again.
- Correct answer are highlighted in green color.
- Wrong answer highlighted in red color.
- Your score update automatically.

-------

## Technology

1. index.html → main layout with three screens.

2. style.css → styling for screens, buttons, and progress bar.

3. script.js → game logic (questions, scoring, screen toggling).
